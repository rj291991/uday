React Enviroment Setup & Run

1. Install Node Modules to run this App: -- npx create-react-app my-app
2. Past & Replace all file in the folder except Node Modules Folder.
3. As I'm using bootstrap and Router - So you need to install these package or you can check all packages in package.json
2. Start app by using this command - >      npm start
